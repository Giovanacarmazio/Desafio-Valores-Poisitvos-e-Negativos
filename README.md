# Positive and Negative Values Challenge

Este repositório contém o projeto **Positive and Negative Values Challenge**, que tem como objetivo processar uma lista de números e identificar quais são positivos e quais são negativos. O sistema permite separar e contar os valores positivos e negativos de forma eficiente.

## Descrição do Projeto

O **Positive and Negative Values Challenge** foi desenvolvido para realizar a análise de uma sequência de números, classificando-os como positivos ou negativos e gerando relatórios com os resultados.

## Funcionalidades

- Análise de valores positivos e negativos.
- Separação e contagem de números em ambas as categorias.
- Geração de relatório sobre a distribuição dos valores.

## Tecnologias Utilizadas

- **Python**

## Estrutura do Repositório

- `values_analysis.py`: Código principal que realiza a separação e contagem dos valores.
- `docs/`: Documentação sobre o projeto e seu uso.

## Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/Giovanacarmazio/Desafio-Valores-Positivos-e-Negativos.git
   ```

2. Execute o script:
   ```bash
   python values_analysis.py
   ```

3. Insira a lista de números para análise e visualize o relatório gerado.
